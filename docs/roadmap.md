# Roadmap

## Status atual

A base técnica principal está concluída:

- autenticação e escopo de usuário
- módulos LiveView de finanças (`/finances`) e tarefas (`/tasks`)
- analytics com cache OTP
- fluxo de finanças compartilhadas entre contas
- suíte de testes extensa (unit, web e property-based)

## Frentes de evolução

## 1. Produto

- Expandir fluxos colaborativos de finanças compartilhadas
- Melhorar experiência mobile para operações recorrentes
- Evoluir onboarding para cenários avançados

## 2. Qualidade

- Reforçar testes de regressão visual
- Automatizar auditorias de acessibilidade e performance no CI
- Expandir cobertura de casos extremos nos property tests

## 3. Operação

- Fortalecer runbook de deploy/rollback
- Aprimorar observabilidade de eventos críticos
- Evoluir estratégias de backup e recuperação

## Atualização de roadmap

Sempre que uma feature relevante for entregue ou o plano mudar, atualize este arquivo no mesmo PR.
